Variable

variable is a container and stores a values in order to use over and over in our code.

Variable Data Types

  • Number : Floating point numbers , for decimals and integers
  • String : Sequence of characters for text
  • Boolean : Logical data type that can be Trueor False
  • Undefined : A variable does not have a value yet
  • Null : non-existent
